# Project Outline

Decided on by our first meeting, but subject to change

# Functionality

What should our app do?

* Use Flashcards
  * Create flashcards
  * Modify flashcards
  * Delete flashcards

* Log in

* Stretch goals
  * Chat
  * Refresh without reload
  * Making it pretty
  * Notifications/reminders(combined with calendar)
  * Share flashcard sets
  * Upvote/Downvote

* Log in page
